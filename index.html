Introduction.

Welcome to the course material for the DIBI 2012 Workshop, Introduction to Ruby on Rails. This contains a handy setup guide, to be completed before the day of the workshop, a run through of some of the important concepts that will be covered, and links off to external resources that are useful for more information and may go into a little more depth.

0. Setup a Rails Development Environment

Setting up your environment can be difficult when beginning Ruby on Rails. There‘s a lot to take on board before you even get started. To make things easy for you, and to ensure that you have everything you need to follow along with the workshop, we have created a Virtual Machine image with the correct setup using popular, open-source tools. You’ll need to download VirtualBox and Vagrant so that you can use our Vagrant Box at the workshop.

  • Download VirtualBox 4.1.12 for your platform and install it.
  • Download Vagrant 1.0.2 for your platform and install it.
    More detailed instructions are available in the Vagrant Documentation

Doing It Yourself

If you’re a little more experienced and want to set up a Rails development environment natively, then we can recommend the JumpstartLab Environment Setup (all platforms) or the Thoughtbot Laptop Script (Mac OSX only). We’re using Ruby 1.9.3 and Rails 3.1.1 on the day, but anything from Ruby 1.9.2 and Rails 3.1 upwards will be fine. Have a look at how we setup Ubuntu for the workshop.

Text Editor

The Rails community eschew IDEs in favour of text-editors. The most popular editors are Vim and Textmate. For this course we recommend Sublime Text 2 because it works the same across Mac, Windows, and Linux, and has an unlimited trial.
