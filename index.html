Introduction.

Welcome to the course material for the DIBI 2012 Workshop, Introduction to Ruby on Rails. This contains a handy setup guide, to be completed before the day of the workshop, a run through of some of the important concepts that will be covered, and links off to external resources that are useful for more information and may go into a little more depth.

0. Setup a Rails Development Environment

Setting up your environment can be difficult when starting with Ruby on Rails. There‘s a lot to take on board before you even get started. To make things easy for you, and to ensure that you have everything you need to follow along with the workshop, we have created a Virtual Machine image with the correct setup using popular, open-source tools. You’ll need to download VirtualBox, Vagrant and a Vagrant Box that we have created. Don’t worry, this is quite straight forward and is designed to work on all major platforms: Windows, Mac OS, and Linux.

  • Download VirtualBox 4.1.12 for your platform and install it.
  • Download Vagrant 1.0.2 for your platform and install it.
    More detailed instructions are available in the Vagrant Documentation

You should now be ready to set up your box. To do so, run the following commands in Terminal or your command prompt. It’s a good idea to create a new directory/folder to work from.

Beginners Tip: Type each line at your prompt (without the $) and hit the Return Key.

$ vagrant box add dibirails /path/to/box
$ vagrant init dibirails
$ vagrant up

This should fire up a virtual machine in the background running our Rails development environment in Ubuntu. You can test that everything is working properly by logging into the machine:

$ vagrant ssh
$ cd sample_app
$ rails s

Visit http://localhost:3000/ in your browser and you should see the welcome page for Michael Hartl’s popular Rails Tutorial. If there are any problems then open a GitHub Issue on the dibirails repository and we’ll try and work through it with you.

Text Editor

The Rails community eschew IDEs in favour of text-editors. The most popular editors are Vim and Textmate. For this course we recommend Sublime Text 2. It works the same across Mac, Windows, and Linux, and has an unlimited trial.
